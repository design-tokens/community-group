Abstract

This specification describes a JSON schema for combining multiple Design Tokens Format JSON files into a flattened collection of Tokens.

This is a public copy of the editors’ draft. It is provided for discussion only and may change at any moment. It is the product of a W3C Community group and is not affiliated with the W3C directly. Don’t cite this document other than as work in progress.
